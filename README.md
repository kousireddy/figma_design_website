# Travel Website

A modern and responsive Travel Website designed in Figma and developed using HTML & CSS.

This project focuses on creating a clean UI, smooth layout structure, and visually appealing travel-themed pages.

### Design Process

UI/UX designed using Figma

Structured layout planning before development

Focused on:

Clean typography

Balanced spacing

Modern color palette

Simple and user-friendly interface

Technologies Used

HTML5

CSS3

Flexbox

Responsive Design Techniques

Pages Included
1. Landing Page

Attractive hero section

Call-to-action buttons

Introduction to travel services

2. Signup Page

User registration form

Clean and minimal layout

Input validation structure

3. Login Page

Simple login form

Styled input fields

Responsive alignment

4. Explore Page

Travel destinations overview

Cards layout for places

Easy navigation to city details

5.Explore City Pages

Detailed view of selected city

Images and description

Booking option

6. Booked Page

Booking confirmation screen

Success indicator (✔)

Navigation back to explore page

### Features

Fully responsive layout

Clean and organized file structure

Easy to customize

Beginner-friendly code structure
